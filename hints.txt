подтягиваем интерпретатор из локлаьного виртуального окружения

pip/pip3 install -r requirements.txt - Для установки необхоимых пакетов из списка

init start: сервер и клиент - один и тот же компьютер, оправка и обработка запросов

8000 - порт, выделенный под приложение в системе

вьюхи - контроллеры в терминологии MVC

python manage.py startapp appname - создание подприложения в рамках 1-ого проекта

view - handler http-запроса


request arg в хендлере – это ссылка на экземпляр класса HttpRequest, который содержит информацию о запросе, о сессии, о куках и так далее. То есть, через переменную request нам доступна вся возможная информация в рамках текущего запроса.
